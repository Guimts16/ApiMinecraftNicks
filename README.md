# API 🛠
Simples de utilizar. Este código irá buscar nicks de 3 letras (Pode ser configurado no código) aleatoriamente no site da Mojang. Para isso, basta:
- Abra o CMD na pasta;
- Digite: node nicks.js
E pronto! Irá começar a procura.
